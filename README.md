### 🚀 How to Run:

1. **Quick Start** (no setup required):
   ```bash
   python3 -m http.server 8000
   ```
   Then visit: `http://localhost:8000`

2. **Live Development** (VS Code):
   - Install Live Server extension
   - Right-click `index.html` → "Open with Live Server"
